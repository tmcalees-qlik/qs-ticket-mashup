Qlik Chart Integration

The visualizations to the right (or below) are served by the Qlik Sense Enterprise server using our Capabilities API. This is a <div> level integration to your web application. The code to complete this integration looks something like this:

// Open a Qlik Sense application var app = qlik.openApp('aa72ba66-9cb1-469d-af42-2a71d485550f', config); // Get a Qlik Sense visualization (object) and place it in the <div> // container with an id of chart1 this.app.getObject('chart1','krWxx'); this.app.getObject('chart2','KYSaNka');
